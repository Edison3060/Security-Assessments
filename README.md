# Security Assessments & Penetration Testing Reports

This repository contains sanitized technical reports from various vulnerability assessments and incident investigations.

* **WebGoat Infrastructure Enumeration:** Documentation of OpenSSH RCE and Slowloris DoS vulnerabilities using Nmap and Nikto.
* **SecureBank Customer Portal:** Exploitation of an undocumented API endpoint via UNION-based SQL Injection to extract database schemas.
* **Support Ticket System:** Hijacking an automated administrative session via Stored XSS for data exfiltration.
* **CryptoLocker Threat Investigation:** Static analysis of ransomware using capa, strings, and regshot to map registry persistence and C2 communication.
